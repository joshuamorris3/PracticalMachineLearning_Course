Practical Machine Learning Course Writeup

My course submission...

  • Cross validation - since we can't use the test set for modelling, use the training set by breaking it up into training and test sets
    • Use 60% training, 20% testing, 20% validation
  • Will we need to remove zero covariates?
  • PCA - use to reduce the number of predictors (variables)
  • Trees
    • When you have a large number of variables
    • perhaps useful to predict with trees (better in nonlinear settings)
    • Without cross validation can lead to overfitting
  • Bagging - bootstrap aggregating
    • Resample data (training data) with replacement (bootstrap)
    • Recalculate prediction function on resampled data
    • average predictors
    • similar bias found
    • less variance (smoother model)
    • Useful for nonlinear models e.g. trees, random forests
  • Random Forests - extension to bagging for classification or regression trees
    • Resample training data with replacement (bootstrap)
    • Rebuild tree on each resampled data
    • As an individual observation goes through each tree it could end up with slightly different prediction
    • Then average the predictions together from each tree
    • Top performing (predicting,very accurate) along with boosting
